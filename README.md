# Password Strength Checker

A simple web application that evaluates password strength in real time.
The tool analyzes the password entered by the user and provides visual feedback about how strong or secure the password is.

## Features

* Real-time password strength detection
* Strength indicator bar
* Checks for:

  * Password length
  * Uppercase letters
  * Numbers
  * Special characters
* Visual feedback with color changes based on password strength

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

The application checks several conditions while the user types a password:

1. Minimum password length
2. Presence of uppercase letters
3. Presence of numbers
4. Presence of special characters

Based on these checks, the password is classified into different strength levels such as **Weak**, **Medium**, **Strong**, or **Very Strong**.

## How to Use

1. Open the application in your browser.
2. Enter a password in the input field.
3. The strength indicator will update automatically as you type.

## Project Purpose

This project was created as a practice exercise to understand:

* DOM manipulation in JavaScript
* Event handling
* Basic password validation logic
* Creating simple interactive web interfaces
